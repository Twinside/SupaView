SupaView

SupaView is a tool to help you locate big folders and big files on your hard disk. Every file is represented by a square, which size is proportional to it's size.

You just have to download it and launch to start finding what's taking so much space!

Minimum system requirements

  • Mac OS X Snow Leopard (10.6)
  • Mac Intel 32 or 64 bits
